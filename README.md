# 📝 Flask Todo Application

A simple Todo List web application built using **Flask** and **SQLite**. This project allows users to create, update, and delete tasks through a clean web interface.

## 📚 About the Project

This project was developed during my **2nd Year, 3rd Semester** as part of my journey in learning Python web development using the Flask framework.

The main objective of this project was to understand:
- Flask Framework
- CRUD Operations
- SQLite Database Integration
- HTML Templates (Jinja2)
- Routing in Flask

## 🚀 Features

- ➕ Add new tasks
- 📝 Update existing tasks
- ❌ Delete tasks
- 📅 Store creation date and time
- 💾 SQLite database for data storage
- 🎨 Simple and responsive user interface

## 🛠️ Tech Stack

- Python
- Flask
- Flask-SQLAlchemy
- SQLite
- HTML5
- CSS3
- Jinja2 Templates

## ⚙️ Installation

1. Clone the repository 

```bash
git clone https://github.com/abhishek-mishra14k/Flask.git
```

2. Move into the project folder

```bash
cd Flask
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python app.py
```

5. Open your browser and visit

```
http://127.0.0.1:5000/
```
